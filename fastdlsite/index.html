fastdl.me

(WIP) big fat fastdl for Counter-Strike: Source maps (includes all gamebanana maps)

Directory list of canon/filtered maps | Directory list of all hashed/unfiltered maps


Contact:
  • Email: hello@fastdl.me

Gamebanana is periodically checked (manually) for new maps.
You can find map info dumps used by this website at https://github.com/srcwr/maps-cstrike

How to request maps to be added:
  • Post in the Sourcejump Discord server or bhoptimer Discord server.
  • Send an email.

Questions:
  • Is it okay to scrape & download everything?
    • Yes.

  • Can I use this as my fastdl?
    • Yes, or you can make your maps url redirect to here.

  • Maybe host .bsp files too?
    • Maybe one day so you can rclone/httpdirfs the URL as your "maps" folder.

  • What's the different between /maps/ & /hashed/?
    • /maps/: slightly curated list that uses the canon/correct maps and removes some corrupted maps.
      • Canon maps csv
      • Removed maps csvs
    • /hashed/: list of everything available (old versions, corrupted files, etc)
Other things:
  • If you want to download with wget/curl:
    • wget https://main.x.fastdl.me/maps/bhop_badges.bsp.bz2
    • curl -L -O https://main.x.fastdl.me/maps/bhop_badges.bsp.bz2

  • Every .nav file is currently a dummy .nav file.
    I'll add the real .nav files where available soon:tm:.

  • I rename maps sometimes when different people reuse the same mapname.
    See renamed_by_me.csv / removed_because_i_renamed_them.csv
